# Plano Completo -- Portfólio Premium (Estilo Telha Clarke)

## Visão Geral

Construir um site de portfólio premium com: - Grid editorial - Animações
suaves - Scroll fluido (Lenis) - Transições elegantes - Performance
otimizada

------------------------------------------------------------------------

## Stack

-   Next.js (App Router)
-   TypeScript
-   GSAP + ScrollTrigger
-   Lenis
-   SCSS ou Tailwind
-   next/image
